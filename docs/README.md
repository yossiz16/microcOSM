# MicrocOSM

This repo contains:

* Containerized OSM tools

* Deployment of the OSM ecosystem using [MicrocOSM](/microcOSM/README.md)
  
* Synchronization process of 2 OSM stacks
  * using diff files.
  * using diff pipeline(based on osmosis) between two 2 databases.
